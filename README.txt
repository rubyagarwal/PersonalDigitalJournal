To run the application, following tools are required.
1. JAVA SDK/EE
2. Eclipse IDE
3. Mongo DB
4. Apache Tomcat Server
5. Web Browser

Steps to run the project:
1. Setup the project in Eclipse IDE.
2. Setup Apache Tomcat plugin in Eclipse.
3. Right-click on the project folder and say Run As -> Run on server.
4. This would start Tomcat and the user profile window should then appear.

Testing done on Mozilla Firefox web browser.

P.S. : The different features of the application can be browsed in the Eclipse IDE or the URL can be pasted to any popular web browser for a better look and feel.
